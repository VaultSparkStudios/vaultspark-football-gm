# Franchise Architect: Football

This folder is the product home for the first sport in the Franchise Architect universe.

Current executable surfaces still live at the repo root while the static build is stabilized:
- `public/` - browser pages and client modules
- `src/` - simulation/runtime/domain code
- `scripts/` - build, smoke, and Studio protocol tooling

Move code into this folder only when the football app is green on `playfranchisearchitect.com` and the import paths can be changed in one tested migration.
