Browser-First Beta

VaultSpark Football GM is designed to run in the browser with local save data. Local leagues, settings, and optional challenge data remain in the player's browser unless the player explicitly exports or shares them.

Feedback

The in-game feedback path opens a public GitHub issue draft with public-safe game context. It does not attach personal account data or full save payloads.
